This is an openArchitectureWare example.

You have to build model, edit and editor code of both provided
genmodel-files located in the /model/ directory.
	1. mymodel.genmodel
	2. simple.genmodel
	
Then you can use the workflow and java files bundeled in the
de.cau.cs.kieler.example.oaw.gen plug-in to exploit the example.
