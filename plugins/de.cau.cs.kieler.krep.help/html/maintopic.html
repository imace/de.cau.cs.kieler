KReP EvalBench

The KReP EvalBench (Kiel Reactive Processors Evaluation Bench) is the tool to evaluate the KEP. It can communicate either with a KEP that runs on an FPGA or with a software simulation. It can be used to execute programs, i.e., load programs into the instruction ROM of the KEP, set inputs, trigger tick, read outputs and the length of the last tick. Note: The current status of the EvalBench is a alpha release. Please send any bug-reports or remarks to ctr@informatik.uni-kiel.de
