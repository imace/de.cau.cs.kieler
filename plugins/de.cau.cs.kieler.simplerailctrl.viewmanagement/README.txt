This plug-in implements a view management component for the
simplerailctrl editor and its generated (executable) ptolemy model.
It reacts as an observer DataComponent for the KIEM (KIELER
Execution Manager) project and highlights the active states in
the simplerailctrl editor.

For this plug-in to work you need the following plug-ins:
de.cau.cs.kieler.simplerailctrl
de.cau.cs.kieler.simplerailctrl.sim.ptolemy
de.cau.cs.kieler.sim.kiem
