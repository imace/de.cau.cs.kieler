This is an openArchitectureWare example.

Make sure you have generated the model, edit and editor code
of the de.cau.cs.kieler.example.oaw plug-in.
	
The XtendJava.java file contains the xtend extensions used in the
various /model/*.ext files.

The following are provided:
	1. Model2Text transformation
	2. Model2Model transformation w/ dynamic EMF 
	3. Model2Model transformation w/ static implementation
	4. Model2Model transformation w/ Java escape code
	5. Inplace transformation
